题目一：位 1 的个数

描述：编写一个函数，输入是一个无符号整数（以二进制串的形式）n，返回其二进制表达式中数字位数为 1 的个数（也被称为汉明重量）

题解 1

设i=0，依次递增，最大为整型长度 32 位，将n与 2 的i次方进行与运算，当且仅当n的第i位为1时，运算结果为1，代码如下：

var hammingWeight = function (n) {
  let ret = 0;
  for (let i = 0; i < 32; i++) {
    if (((1 << i) & n) !== 0) {
      // 先将1左移i位，也就是2的i次方，再跟n做与运算
      ret++;
    }
  }
  return ret;
};

复杂度分析：

  • 时间复杂度：O(i)，其中i为int型数n的二进制位数

  • 空间复杂度： O(1)，值需要常数的空间保存若干变量

题解 2

位运算优化，观察规律：n & (n - 1)，其运算结果恰为把 n 的最低位的1置为0的结果，即 4 = 6 & (6 - 1), 0100 = 0110 & 0101 即运算 n & (n - 1)的次数为位 1 出现的次数，代码如下：

var hammingWeight = function (n) {
  let ret = 0;

  while (n) {
    n &= n - 1;
    ret++;
  }
  return ret;
};

复杂度分析：

  • 时间复杂度：O(logN)

  • 空间复杂度：O(1)

题目二：链表相加

描述：两个链表 L1, L2, val 值是正整一位数，现在从表头开始，将两个链表的 val 相加，进位数字加到下一次的相加数上，

如 L1 = [1,2,3], L2 = [4,5,6], 则 L3 = [5,7,9]

若 L1 = [5,5,6], L2 = [4,7,9], 则 L3 = [9,2,6,1]

// 声明链表
function LinkNode(val, next) {
  this.val = val !== undefined ? val : null;
  this.next = next !== undefined ? next : null;
}

// 链表相加
function addNumbers(l1, l2) {
  let head = null; // 头结点
  let tail = null; // 尾结点
  let carry = 0;

  while (l1 || l2) {
    let n1 = l1 ? l1.val : 0;
    let n2 = l2 ? l2.val : 0;
    let sum = n1 + n2 + carry; // 求和

    if (!head) {
      head = tail = new LinkNode(sum % 10); // 头结点赋值
    } else {
      tail.next = new LinkNode(sum % 10); // 尾结点赋值
      tail = tail.next;
    }

    carry = Math.floor(sum / 10); // 是否进位

    if (l1) l1 = l1.next;
    if (l2) l2 = l2.next;
  }
  if (carry) tail.next = new LinkNode(carry); // 若需进位，则增加尾结点

  return head;
}

复杂度分析：

时间复杂度： O(Max(m, n))， m，n 分别为链表 l1，l2 的长度

空间复杂度： O(Max(m, n))， m，n 分别为链表 l1，l2 的长度
